📦 Installation

Clone the repository:

git clone https://github.com/amirebog/Amirog.git

Navigate to the project:

cd Amirog

Install dependencies:

npm install

Run the development server:

npm run dev

Open:

http://localhost:3000

---

🚀 Deployment

Deploy instantly with Vercel:

npm run build
